Usage: algokit completions uninstall [OPTIONS]

  Uninstall shell completions, this command will attempt to update the
  interactive profile script for the current shell to remove any algokit
  completions that have been added. To specify a specific shell use --shell.

Options:
  --shell [bash|zsh]  Specify shell to install algokit completions for.
  -h, --help          Show this message and exit.
