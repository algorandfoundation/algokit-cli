Usage: algokit [OPTIONS] COMMAND [ARGS]...

  AlgoKit is your one-stop shop to develop applications on the Algorand
  blockchain.

Options:
  --version                       Show the version and exit.
  -v, --verbose                   Enable logging of DEBUG messages to the
                                  console.
  --color / --no-color            Force enable or disable of console output
                                  styling.
  --enable-version-check / --disable-version-check
                                  Enable or disable version checking and
                                  reporting permanently.
  --skip-version-check            Skip version checking and reporting.
  -h, --help                      Show this message and exit.

Commands:
  bootstrap    Bootstrap AlgoKit project dependencies.
  completions  Install and Uninstall AlgoKit shell integration.
  doctor       Run the Algorand doctor CLI.
  goal         Run the Algorand goal CLI against the AlgoKit Sandbox.
  init         Initializes a new project.
  sandbox      Manage the AlgoKit sandbox.
